# 🚗 Vehicle Service Booking System

## 📌 Overview

This is a web-based application developed using **Spring Boot, Thymeleaf, and MySQL** that allows users to book vehicle services online.

---

## 🚀 Tech Stack

* Java
* Spring Boot
* Thymeleaf
* Spring Data JPA
* MySQL

---

## 🔥 Features

* User registration and login
* Book vehicle service
* Manage bookings
* Admin service management
* Clean UI using Thymeleaf

---

## ▶️ How to Run

1. Clone the repository
2. Configure MySQL database
3. Run Spring Boot application
4. Access via browser
